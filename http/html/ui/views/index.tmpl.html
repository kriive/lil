what's lil?

lil is a simple URL shortening service written in Go. lil is a project by kriive. it was created to both solve a simple problem and to learn some modern web tooling.

lil's focus is to be simple to use and to use as few services as possible: it runs on a single node, its frontend is written in plain HTML + CSS and it uses SQLite as a database.

can i contribute?

of course! you can find the project here. make sure to check out this project, this is the main inspiration for lil.
